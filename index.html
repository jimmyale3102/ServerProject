Taller de Windows Server r12

Windows Server 2012 es un sistema operativo con todas las funciones para servidor, basado en las mejoras que Microsoft presentó en Windows Server 2008 Release 2. Windows Server 2012 es el sucesor de Windows Server 2008 R2

Descripción

El taller de Windows Server r12, da una pequeña descripción sobre lo más escencial de configuración a momento de instalar y montar los servicios de:

  • Active Directory.
  • Domain Name System (DNS).
  • Dynamic Host Configuration Protocol (DHCP).
  • File Transfer Protocol (FTP).
  • Simple Mail Transfer Protocol (SMTP).

Instalacion de Windows Server r12

Para la instación y desarrollo del taller, se procedio a descargar una Máquina Virtual, en este caso, VMware y obtenida la imagen .iso de sistema operativo, se dio inicio al desarrollo del taller.
La instalación de Windows Server 2012, es muy acorde a las necesidades de cada individuo, en este caso se utilizo la instalacion básica que se encuentra en la Red.
